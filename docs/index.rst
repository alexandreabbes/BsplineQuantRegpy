BsplineQuantRegpy
=================

**BsplineQuantRegpy** - Régression quantile avec B-splines sous contraintes de forme.

Ce package implémente la régression quantile avec des B-splines de degrés 1 à 4,
sous contraintes de forme (monotonie, convexité, dérivée troisième).
