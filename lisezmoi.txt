### Quantile Regression with Constrained Splines

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.6+](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)

Ensemble de programmes pour la régression quantile avec splines sous contraintes, basé sur
l'article  "Quantile regression with cubic splines under shape constraints" par Alexandree Abbes,  https://zenodo.org/records/16999785. 

### Prérequis (la librairie cvxpy est plus difficile à installer)

```bash
pip install pandas numpy scipy warnings matplotlib cvxpy

###Installation
#copier tous les fichiers dans un meme repertoire. 


### Structure des fichiers
#Interface graphique
    Quant_reg_tk.py - Interface graphique Tkinter pour tester interactivement les paramètres (degré des splines, nœuds, contraintes de dérivées)

#Algorithmes principaux

#Chaque fichier est autonome et inclut des tests:
#Fichier	    Description
SplineCubicQuantBspkn3__V2_der3.py	Splines cubiques avec contraintes sur dérivées 1ère, 2ème, 3ème
SplineQuarticQuantBspkn4_V3.py	Splines quartiques avec contraintes exactes sur tout l'intervalle
SplineQuadQuantBspkn2_V1.py	Splines quadratiques avec contraintes sur dérivées 1ère, 2ème
SplineLinQuantBspkn1_V1.py	Splines linéaires avec contraintes sur dérivée 1ère

#Exemples et données
    Test_temp.py - Réplication du test sur les températures globales (données dans temp.xls)



## Utilisation rapide
# Exemple d'utilisation d'un fichier
python3 SplineCubicQuantBspkn3__V2_der3.py
# Exemple d'utilisation globale
python3 Quantregtk.py


 Fonctionnalités

    ✅ Régression quantile avec splines de degré 1 à 4

    ✅ Contraintes sur les dérivées (valables sur tout l'intervalle ou des régions choisies, pas uniquement aux noeuds)

    ✅ Interface graphique pour tests interactifs

    ✅ Code autonome avec tests intégrés
